## Quick Start

### Minimal Configuration Example

```toml
version = "1.0"

[[groups]]
name = "hello_world"

[[groups.commands]]
name = "greet"
cmd = "/bin/echo"
args = ["Hello, World!"]
```

### Basic Usage

1. **Create configuration file**: Create a `config.toml` file
2. **Create hash file** (if needed):
   ```bash
  record -file config.toml
  record -file /bin/echo
   ```
3. **Execute**:
   ```bash
   go-safe-cmd-runner -file config.toml
   ```
